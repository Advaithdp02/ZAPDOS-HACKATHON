# **App Name**: CampusConnect

## Core Features:

- Role-Based Access: Secure login and access control based on user roles (Student, HOD, TPO).
- Drive Management: TPO can create and manage placement drives, including company details and eligibility criteria.
- Student Signup: Students can register, upload their résumés, and view/enroll in active drives.
- Automated Notifications: System sends automated emails to students regarding drive status and results.
- Reporting & Analytics: HOD can generate department-wise, offer-wise, and student-wise reports in Excel/PDF.
- AI Email Generation: AI-powered tool to generate email templates for various stages of the placement process.
- Resume Filtering: AI tool to extract candidate information (skills, experience, education) and filter applications for recruiters. Resume information will be reformatted into professional looking templates, providing uniformity of formatting across applications.

## Style Guidelines:

- Primary color: Deep blue (#34495E) for a professional and trustworthy feel.
- Background color: Light gray (#ECF0F1) for a clean and modern interface.
- Accent color: Soft green (#2ecc71) for positive actions and highlights.
- Body font: 'Inter', a sans-serif font for a clean, modern, neutral feel; will also be used for headlines
- Code font: 'Source Code Pro' for displaying code snippets (e.g., in error messages).
- Use professional and clear icons to represent different functions and data categories.
- Subtle transitions and animations for a smooth and engaging user experience.
